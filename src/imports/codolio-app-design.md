Design a modern web application UI called **Codolio – Developer Coding Portfolio & Analytics Platform**.

The platform aggregates coding stats from LeetCode, CodeChef, and GitHub and provides developers with a portfolio, analytics dashboard, and AI coding assistant.

Design style:

* Modern developer dashboard
* Dark theme with subtle gradients
* Colors: dark background (#0f172a), accent emerald/teal (#10b981), light text (#e5e7eb)
* Minimal, clean UI similar to Linear, Vercel, and GitHub dashboards
* Use rounded cards, soft shadows, and modern typography (Inter or Outfit)

Create the following responsive pages.

---

1. Landing Page

Sections:

* Navbar (Logo, Features, Pricing, Login, Sign Up)
* Hero section with headline:
  “Track Your Coding Journey in One Place”
* Subtext explaining coding profile aggregation
* CTA buttons: “Get Started” and “View Demo”
* Feature section with 4 cards:
  Coding Analytics
  Developer Portfolio
  Contest Tracker
  AI Coding Assistant
* Screenshot preview of dashboard
* Trusted by developers section
* Footer with links and social icons
